# Tighogg - terminal fighting game

This is a fighting game for the terminal inspired by
[Nighogg](http://nidhogggame.com/). Each player's goal is to reach their and of
the map.

## Controls

### Player 1

-	Left, Right - move
-	Down - Stop
-	Up - Jump

### Player 2

-	a, d - move
-	s - Stop
-	w - Jump
